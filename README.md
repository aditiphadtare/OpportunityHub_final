# Opportunity Hub

A comprehensive platform for students to find hackathons, tech events, internships, and jobs.

## Project Structure

- **frontend/**: React/Vite/Shadcn UI application.
- **backend/**: Node.js/Express application refactored to industry standards.

## Prerequisites

- Node.js (v18+)
- Firebase Project
- Gemini API Key

## Setup

### Backend
1. `cd backend`
2. `npm install`
3. Create `.env` from `.env.example`
4. Place `firebaseKey.json` in `backend/`
5. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. Create `.env` from `.env.example`
4. `npm run dev`

## Features

- **Domain & Location Preferences**: Tailored opportunity feed.
- **AI Resume Analysis**: Match percentage and skill improvement suggestions.
- **Deadline Tracking**: Wishlist with countdowns for upcoming deadlines.
